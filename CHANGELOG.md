# Changelog

所有 notable 变更都会记录在这个文件中。

## [1.1.0] - 2026-06-20

### 新增

- 设置页面：开始菜单点击 ⚙️ 进入，支持游戏模式、按键绑定、游戏参数、音频、网络、语言设置
- 单人 / 双人模式切换：默认单人模式，无需服务器；双人模式自动连接 WebSocket
- 按键自定义：可在设置中修改各个操作的键盘按键
- 音量调节：通过 Web Audio gain 节点实时控制音效音量
- 落地阴影开关：可在设置中开启或关闭
- 语言选择：支持中文 / English / 跟随系统（刷新生效）
- 设置状态持久化：所有设置自动保存到 localStorage

### 调整

- 游戏结束判定改为“下一个方块无法放入出生位置才算失败”，允许方块触顶后继续游戏
- 初始速度档位上限改为最高等级（L14）
- 单人模式游戏结束后可直接按 `R` 重新开始
- 虚拟键盘按钮事件绑定到外层 div，提升点击响应区域
- 设置页开关改为滑块样式，下拉框改为自定义样式

### 修复

- 修复单人模式死亡后无法按 `R` 重开的问题
- 修复初始速度只能调到 L6 的问题

## [1.0.1] - 2024-06

### 新增

- 双人 WebSocket 在线对战模式
- 房间系统：通过 URL 房间号进入同一房间
- 分屏显示本地与远端玩家画面
- 对战计分板：实时显示双方分数、等级、时间与状态
- 加时赛机制：一方死亡后另一方进入追分阶段
- 游戏结果判定：根据分数与存活时间判定胜负
- 方块落地阴影提示
- 保留原项目的单人模式

### 调整

- 顶部房间信息区改为分层布局，避免遮挡标题
- 页面标题与 meta 信息改为 `Tetris Online - 俄罗斯方块在线对战`
